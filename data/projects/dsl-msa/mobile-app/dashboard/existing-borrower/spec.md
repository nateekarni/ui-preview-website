# Dashboard ผู้กู้ยืมเดิม (MOB-DASH-001)

## วัตถุประสงค์
หน้าหลักของแอปสำหรับผู้กู้ยืมที่มีบัญชีอยู่แล้วกับ กยศ. หรือ กรอ. แสดงภาพรวมบัญชีและยอดหนี้แบบ Real-time

## ข้อมูลที่แสดง

### Summary Section
- ยอดเงินต้นคงเหลือรวมทุกบัญชี (฿249,846.19 จากยอดกู้รวม ฿296,400)
- ความคืบหน้าการชำระ (35.3% = ฿136,553.81 ชำระแล้ว)
- ยอดที่ต้องชำระรวม (฿12,103.83) พร้อมปุ่มชำระทั้งหมด

### Loan Account List
รายการบัญชีแบบ Card แต่ละบัญชีประกอบด้วย:
- ประเภทบัญชี (กยศ. / กรอ.) + Tag สถานะ (ปกติ / ค้างชำระ / ปิดบัญชี)
- หมายเลขบัญชี
- ยอดคงเหลือสุทธิ
- ยอดงวดที่ต้องจ่าย + วันครบกำหนด
- ยอดหนี้ค้างสะสม (ถ้ามี)
- ปุ่ม "รายละเอียดบัญชี" และ "ชำระเงิน"

### Quick Utilities
- สายด่วน กยศ. (โทร 020164888)
- ส่งอีเมลติดต่อ
- LINE Official

## Navigation / Flows
แอปมี Bottom Navigation bar สำหรับสลับ View:
- Dashboard (ผู้กู้เดิม) → MOB-DASH-001
- ลงทะเบียนกู้ใหม่ → MOB-NEWLOAN-001
- ผู้ค้ำประกัน → MOB-GUARANTOR-001
- บริการสถาบัน → MOB-INST-001
- บริการสาธารณะ → MOB-GENERAL-001

## Multi-Account Selector
Dropdown ที่ Header สำหรับสลับบัญชีหลัก (กยศ. #100825401 / กรอ. #10082542 / กยศ. #100825399)

## Design Notes
- Mobile-first, รองรับหน้าจอ 390px
- ใช้ Color coding: emerald = ปกติ, rose = ค้างชำระ, slate = ปิดบัญชี
- Dynamic Island + Status bar จำลองบน Phone Shell
- Portal web view (Desktop) สำหรับเปิด preview และเลือก Flow
