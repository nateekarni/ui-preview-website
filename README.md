# UX/UI Preview Hub

Foundation scaffold for an internal UX/UI preview hub.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- File-based project and screen metadata

## Useful commands

```bash
pnpm install
pnpm dev
pnpm test
pnpm build
```

## Data layout

Project and screen source files live under `data/projects`. HTML previews live under `public/previews`, with shared preview styling in `public/preview-assets/design-system.css`.
